# Google Sheets 題庫模板指南

## 📋 快速設定

### 步驟 1：建立新試算表

1. 前往 [Google Sheets](https://sheets.google.com)
2. 點擊「建立新試算表」
3. 命名為 `法律考題題庫` 或你喜歡的名稱

### 步驟 2：設定欄位

在第一列設定以下欄位（**欄位名稱必須完全相同**）：

```
A1: ID
B1: 類型
C1: 科目
D1: 題目內容
E1: 參考解答
F1: 分數
```

## 📝 欄位說明

| 欄位 | 類型 | 必填 | 說明 | 範例 |
|------|------|------|------|------|
| **ID** | 文字 | ✅ | 題目唯一編號 | Q001, Q002, Q003 |
| **類型** | 文字 | ✅ | 題目類型 | 申論題, 實例題, 選擇題 |
| **科目** | 文字 | ✅ | 法律科目 | 民法, 刑法, 民訴, 刑訴, 行政法, 憲法 |
| **題目內容** | 文字 | ✅ | 完整題目敘述 | 甲向乙購買一輛機車，約定於三日後交付... |
| **參考解答** | 文字 | ❌ | 標準答案或解析 | 本題涉及民法物權編關於動產所有權之移轉... |
| **分數** | 數字 | ✅ | 題目配分 | 25, 50, 100 |

## 🎯 最佳實踐

### 1. 題目編號規則

建議使用統一的編號規則，便於管理：

```
Q001 - 民法第1題
Q002 - 民法第2題
Q101 - 刑法第1題
Q102 - 刑法第2題
Q201 - 民訴第1題
```

### 2. 科目命名規範

保持一致的科目名稱（避免「民法」和「民 法」混用）：

```
✅ 推薦
- 民法
- 刑法
- 民事訴訟法
- 刑事訴訟法
- 行政法
- 憲法
- 商事法

❌ 避免
- 民 法（有空格）
- 民法 （尾部有空格）
- 民法(大陸法)（括號混用）
```

### 3. 題型命名規範

```
✅ 推薦
- 申論題
- 實例題
- 選擇題
- 混合題

❌ 避免
- 申論
- 論文
- 案例分析
```

### 4. 分數設定

建議使用標準分數：

```
✅ 推薦
- 25 分（1/4 份量）
- 50 分（1/2 份量）
- 100 分（標準題）

❌ 避免
- 23 分（不易組合）
- 17 分（不易組合）
```

## 📥 匯入現有題目

### 從 Excel 匯入

1. 在 Google Sheets 中點擊「檔案」→「匯入」
2. 選擇「上傳」
3. 選擇你的 Excel 檔案
4. 選擇「建立新試算表」或「插入新工作表」
5. 點擊「匯入」

### 從 CSV 匯入

1. 準備 CSV 檔案，欄位順序為：ID, 類型, 科目, 題目內容, 參考解答, 分數
2. 在 Google Sheets 中點擊「檔案」→「匯入」
3. 上傳 CSV 檔案
4. 選擇「建立新試算表」
5. 點擊「匯入」

## 🔗 分享試算表

### 步驟 1：設定分享權限

1. 點擊右上角「分享」按鈕
2. 在「誰可以存取」中選擇「任何有連結的人」
3. 確保權限設為「檢視者」（不需要編輯權限）
4. 點擊「複製連結」

### 步驟 2：取得試算表 ID

從分享連結中提取 ID：

```
原始連結：
https://docs.google.com/spreadsheets/d/1a2b3c4d5e6f7g8h9i0j/edit?usp=sharing

試算表 ID：
1a2b3c4d5e6f7g8h9i0j
```

## 🔄 定期維護

### 備份題庫

1. 點擊「檔案」→「下載」
2. 選擇「逗號分隔值 (.csv)」
3. 定期保存備份

### 版本歷史

Google Sheets 自動保存所有版本：

1. 點擊「檔案」→「版本歷史」
2. 查看或恢復之前的版本

## 📱 行動裝置編輯

Google Sheets 支援行動裝置編輯：

1. 在手機上打開 Google Sheets 應用
2. 打開你的試算表
3. 隨時隨地新增或修改題目
4. 自動同步到雲端

## 🎨 格式化建議

### 凍結第一列

1. 選擇第二列（點擊「2」）
2. 點擊「檢視」→「凍結」→「凍結 1 列」
3. 便於滾動時保持欄位標題可見

### 設定欄寬

1. 雙擊欄位邊界自動調整寬度
2. 或手動拖動調整

### 使用顏色區分

1. 選擇第一列
2. 點擊「填滿顏色」圖示
3. 選擇淺色背景，便於識別

## 🚀 進階技巧

### 資料驗證

為「科目」欄位設定下拉選單：

1. 選擇「科目」欄位（C2:C1000）
2. 點擊「資料」→「資料驗證」
3. 選擇「清單」
4. 輸入選項：民法, 刑法, 民訴, 刑訴, 行政法, 憲法
5. 點擊「儲存」

### 條件格式

為空白的「參考解答」欄位設定警告：

1. 選擇「參考解答」欄位（E2:E1000）
2. 點擊「格式」→「條件格式」
3. 選擇「是空白」
4. 選擇警告顏色（如淺紅色）
5. 點擊「完成」

## 📊 統計與分析

### 計算題目數量

在空白儲存格輸入：
```
=COUNTA(A2:A)
```

### 計算總分

在空白儲存格輸入：
```
=SUM(F2:F)
```

### 按科目統計

在空白儲存格輸入：
```
=COUNTIF(C2:C, "民法")
```
